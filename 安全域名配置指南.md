# CloudBase 安全域名配置指南

**配置时间：** 2026-04-01  
**环境ID：** `bluetooth-website-3eyhlocfe7b405`

---

## 🎯 配置目标

添加开发环境域名：`http://localhost:3001`

---

## 📋 详细配置步骤

### Step 1: 进入安全配置页面

**访问地址：**  
https://tcb.cloud.tencent.com/dev?envId=bluetooth-website-3eyhlocfe7b405#/env/security

**或者通过菜单：**
```
左侧菜单
└── 环境配置
    └── 安全管控 ← 点击这里
```

---

### Step 2: 找到 Web SDK 安全域名区域

在安全配置页面，你会看到几个区域：

```
安全管控
├── [Web SDK 安全域名] ← 找到这个区域
├── 云函数安全域名
├── 云存储安全域名
└── 匿名登录状态有效期
```

---

### Step 3: 添加域名

#### 3.1 点击"添加域名"按钮
- 通常是一个 **"+ 添加域名"** 按钮或 **"+"** 图标
- 位置在 Web SDK 安全域名区域的右上角或底部

#### 3.2 输入域名信息
在弹出的对话框中：

**域名类型：** 选择 `Web SDK 安全域名`

**域名：** 输入 `http://localhost:3001`

**说明（可选）：** 输入 `本地开发环境`

#### 3.3 确认添加
点击 **"确定"** 或 **"保存"** 按钮

---

### Step 4: 验证配置成功

配置成功后，你应该看到：

```
Web SDK 安全域名
├── http://localhost:3001  ✓ 已启用
└── （添加其他域名按钮）
```

## ⚠️ 常见问题

### Q1: 找不到"安全管控"菜单？

**解决方案：**
1. 确保你已经登录腾讯云账号
2. 确保你选择了正确的环境（bluetooth-website-3eyhlocfe7b405）
3. 尝试直接访问链接：
   https://tcb.cloud.tencent.com/dev?envId=bluetooth-website-3eyhlocfe7b405#/env/security

---

### Q2: 添加域名后显示"域名格式错误"？

**解决方案：**
- 确保输入完整的URL：`http://localhost:3001`（包含 http://）
- 不要包含路径（如 `/api`）
- 不要包含末尾斜杠（如 `http://localhost:3001/`）

---

### Q3: 添加域名后显示"域名已存在"？

**说明：** 域名已经配置过了，无需重复添加。

---

### Q4: 配置后还是无法连接CloudBase？

**排查步骤：**
1. 确认开发服务器运行在 3001 端口
2. 清除浏览器缓存
3. 重启开发服务器
4. 检查 `.env.local` 文件中的环境ID是否正确
